# Project: Football Predictor AI - Frontend

## Objective
Create a deluxe, luxury frontend web application that displays daily matches and their probabilistic predictions, driven by the existing Python backend.

## Checklists

### Issue 6: Deluxe Frontend Application
- [ ] Set up a REST API (e.g., FastAPI) to expose the existing predictor logic.
- [ ] Set up a modern frontend framework (e.g., Next.js, Vite/React, or vanilla HTML/CSS if preferred by the user, but we will likely use React/Vite for a dynamic app).
- [ ] Design a premium UI system (dark mode, glassmorphism, gold accents).
- [ ] Fetch today's fixtures from the API-Football v3 API (via the backend) and display them in a grid/list.
- [ ] Implement interactive components where clicking a match fetches and displays the `MatchFactorReport` and LLM-formatted prose.
- [ ] Ensure mobile responsiveness and smooth animations.
